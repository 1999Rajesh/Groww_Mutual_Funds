# Project Scripts

Executable Python scripts for the RAG Mutual Funds project.

## Data Loading Scripts
- `load_sample_data.py` - Load sample mutual fund data into ChromaDB

## Phase Runners
- `run_phase1.py` - Execute Phase 1 implementation
- `run_phase2.py` - Execute Phase 2 implementation
- `run_phase3.py` - Execute Phase 3 implementation
- `run_phase4.py` - Execute Phase 4 implementation
- `run_phase4_5.py` - Execute Phase 4.5 implementation
- `run_phase6.py` - Execute Phase 6 implementation
- `run_phase7.py` - Execute Phase 7 implementation
- `run_phase8.py` - Execute Phase 8 implementation
- `run_phase9.py` - Execute Phase 9 implementation

## Application Startup
- `start_all.py` - Start both backend and frontend servers together
- `start_app.py` - Start the application

## Scrapers
- `run_scraper.py` - Run web scrapers for data collection
- `scraper_runner.py` - Alternative scraper runner

## Usage

```bash
# Activate virtual environment
.venv\Scripts\activate

# Run a specific phase
python scripts/run_phase7.py

# Load sample data
python scripts/load_sample_data.py

# Start all services
python scripts/start_all.py
```
